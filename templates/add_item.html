User Guide

Follow the steps below to add an item for auction:

  • Enter the Title of the item in the first input field.
  • Provide a detailed Description of the item in the textarea.
  • Set the Starting Bid amount in the respective input field.
  • Upload an Image of the item by clicking the "Choose File" button.
  • Select the Auction End Time using the date and time picker.
  • Click the Add Item button to submit the form and add the item to the auction.
  • To return to the home page, click the Back to Home button.
